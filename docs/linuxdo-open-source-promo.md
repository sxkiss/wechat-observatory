## 项目介绍正文

由于微信的大部分纯协议机器人已经无法使用，也很容易被风控，所以我做了一个 LSPosed 模块，通过 hook 的方式直接调用微信 App。

简单说，就是直接操控手机上已经登录的微信，而且可以同时操控多台设备。

### 重要前提

1. 需要解 BL 锁。小米用户我已经在项目 README 里指路了。
2. 安装 Magisk 和 LSPosed 框架。
3. 安装“隐藏应用列表”模块并完成配置。
4. 安装“隐藏解 BL 锁”模块。不安装也可以，但看起来会影响某些功能，比如人脸身份认证。

### 当前情况

- 老号半个月未被风控
- 新号使用一周未被风控

但我并没有实现 AI 机器人，因为现在写出来的基础框架还有其他用途。如果各位想实现自己的微信机器人，还是需要自己继续开发。
